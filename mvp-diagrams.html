For Volodymyr · April 2026

MVP 1 — AI Job Documentation Tool
Crew photos → structured client report + SEO images + CRM log + SDS record. Automatically.
📸
Crew
Takes photos on job. Sends to Telegram bot with captions
→
🤖
Telegram Bot
Receives photos + captions. Triggers processing pipeline
→
🧠
Claude API
Reads images. Generates SEO filename + alt text. Structures report data
→
📄
PDF Generator
HTML → branded PDF report. Areas cleaned, photos, issues, products, SDS refs
→
📤
Delivery
PDF sent to FM. Images + report filed in Drive. Visit logged in PipeDrive
📧 To FM client
Branded PDF cleaning report — same day as the visit
🗂 Google Drive
Report + SEO-named images filed in client folder
📊 PipeDrive CRM
Visit activity logged against client record
🧪 SDS Record
Products used linked to SDS sheets on file per client
🌐 Website / Social
SEO-optimised images ready to use — zero extra work
MVP 2 — AI Estimation Tool
Walk-through data in → branded proposal PDF out → prospect notified → CRM deal created. Under 5 minutes.
🚶
Walk-Through
Owner inputs: sq ft, facility type, scope, frequency, special requirements
→
📱
Input Interface
Telegram bot or simple web form. Rate card loaded once per client
→
🧠
Claude API
Generates scope of work, staffing, compliance inclusions, pricing
→
📋
PDF Generator
Branded proposal: scope, schedule, pricing table, compliance section, signature block
→
📤
Delivery
PDF to prospect + owner. Filed in Drive. CRM deal created in PipeDrive
📧 To prospect
Professional branded proposal PDF — in under 5 minutes
🗂 Google Drive
Proposal stored per client/per estimate for reference
📊 PipeDrive CRM
Deal created automatically with prospect details + proposal attached
⏱ Time saved
3–4 hours per estimate → under 5 minutes. 10–20 estimates/month = weeks saved
